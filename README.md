# Курсовой проект: "Выявление людей с сердечно-сосудистыми заболеваниями"
1) Проект разработан с целью выявления сердечно-сосудистых заболеваний у людей(пациентов) разных возрастов и пола. Кроме того для выявления следующих болезней используются дополнительные признаки, которые человек вводит на основе своих анализов и заключений врача. Все данные, собранные в одну таблицу(dataset), анализируются, на их основе обучается модель при помощи различных методов машинного обучения. По итогу всех тестов и обучений была найдена модель, которая дает наиболее приближенный к истине результат с точностью до 87%.
2) Затем проект был расширен до написания клиент-сервера на основе бота telegram, которая является грубо говоря формой, которую должен заполнить пользователь(пациент). Был написано и модифицирован интерфейс telegram-бота. Это сделало пользованием приложением намного удобнее и понятнее(добавлены кнопки, запросы вопрос-ответ и т.д.)
3) По итогу, результатом заполнения формы является класс, к которому будет причисляться то или иной человек, в зависимости от указанных им признаков. Всего 2 класса: 1. Человек, имеющий сердечно-сосудистые заболевание; 2. Человек без признаков сердечно-сосудистых заболеваний.
Заключение: Разработка данного проекта дала много интересного опыта и знаний по работе с данными, их анализу, и кроме того дала свои плоды в начинаниях по новым для меня областям - машинному обучению и работе в среде разработки python
